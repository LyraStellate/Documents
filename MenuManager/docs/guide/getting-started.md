# 導入方法

Lyra Menu Managerの導入手順について

## 必須アセットの確認

本ツールの動作には、以下のツールが事前にプロジェクトへインポートされている必要があります。

| アセット | リンク |
|----------|--------|
| **Modular Avatar** | [公式サイト](https://modular-avatar.nadena.dev/ja/) |
| **NDMF** (Non-Destructive Modular Framework) | [GitHub](https://github.com/bdunderscore/ndmf) |

::: tip 💡 ヒント
Modular AvatarをVCCからインストールすると、NDMFも自動的にインストールされます。
:::

## インストール手順

### VCCからインストール

準備中

1. ダウンロードした`LyraStellateVPM`をダブルクリックして起動し、VCCまたはALCOMに登録します。
2. インストールするアバタープロジェクトのパッケージリストから **Lyra Menu Manager** を探し、最新バージョンを追加します。
3. Unityエディタを開くと、ツールが利用可能になります。


## 認証方法

有料版をご購入いただいた方は、`Lyra Menu Manager` > `Auth`から認証ウィンドウを開き、BOOTHの注文番号を入力して認証してください。

### 注文番号の確認方法

BOOTHの注文番号が分からない方は、以下の手順でご確認ください。

1. BOOTHの[購入履歴](https://accounts.booth.pm/orders)に移動し、Lyra Menu Managerの注文詳細を開く
2. 注文詳細の`注文番号：xxxxxxxx`となっている数字のみをコピーし、認証ウィンドウに貼り付けてください。

正しい注文番号を入力してもエラーになってしまう場合は、お手数ですが下部フォールダウンから認証申請を送信して下さい。
